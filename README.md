# Mbtmi

## 서비스 개요

mbti만 공개하며 소통할 수 있는 익명 커뮤니티<br>
기본적인 커뮤니티 기능을 포함하여<br>
mbti별 투표 통계, 밸런스 게임 등 특색있는 컨텐츠 제공
